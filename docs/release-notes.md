# HoomNote 0.5.7

- Rebranded the application, package format, executable, local data paths, and
  release artifacts as HoomNote.
- Added automatic migration of existing local libraries into HoomNote storage.
- Connected installed builds to the public HoomNote GitHub Releases OTA feed.
- Added a one-command GitHub release publisher for future updates.
- Added a one-click Windows installer and a portable ZIP to every release build.
- Added installed-app OTA update checks with safe download, autosave, and restart handling.
- Added a manual **Check for updates** action to the workspace menu.
- Added versioned update-feed packages and SHA-256 checksums for distribution.
- Retained the new compact, edge-docked editor command bar.
